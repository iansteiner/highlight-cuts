How to use Highlight Cuts

1. Video File: Select a video file from the data/ directory.

2. Google Sheet URL: Paste the link to your Google Sheet. It must be shared with "Anyone with the link".

3. Game & Player: Select the game and player you want to generate highlights for.

4. Padding: Add extra time (in seconds) before and after each clip.
